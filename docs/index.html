Bem-vindo

Este é O Zen do R, seja bem-vindo! O objetivo deste livro é ensinar ao leitor algumas formas simples de melhorar a organização de seus projetos de análise de dados em R.

O texto deste livro foi escrito por clente com o suporte, ajuda e revisão do time da Curso-R. Ele está disponível gratuitamente no Github sob a licença GNU GPLv3.

Ele foi escrito em RMarkdown com o pacote bookdown.
